# Examples

运行示例：

```bash
npx tsx examples/01-basic-position.ts
npx tsx examples/02-geocentric-state.ts
npx tsx examples/03-sky-observation.ts
npx tsx examples/04-state-vector.ts
```

## 示例列表

| 文件 | 功能 |
|------|------|
| `01-basic-position.ts` | 基本位置查询、坐标系转换 |
| `02-geocentric-state.ts` | 地心视黄道状态、逆行判断、修正对比 |
| `03-sky-observation.ts` | 站心观测、地平坐标、大气折射 |
| `04-state-vector.ts` | 完整状态向量（位置+速度） |
| `05-remote-data.ts` | 加载远程/本地扩展数据 |
| `06-custom-resolver.ts` | 自定义解析器（扩展天体） |
